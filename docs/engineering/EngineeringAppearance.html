    EngineeringAppearance

    Engineering appearance is a map of engineering element class names in kebab case to model element appearance.

    Engineering appearance allows to customize UI generation.

    See default-appearance.yml for the default appearance definition, appearance.yml and getAppearanceLocations() in TestModel for an example of customization.

    Load specification

    Key Type Homogenous Strict containment Description
    model-elements ModelElementAppearanceEntry true

    Mapping of model element EClass names in kebab case to model element appearance. E.g. for IssueStatus appearance specification will look like:

    model-elements:
    issue-status:
        roles: context
