    Load specification

    Key Type Homogenous Strict containment Description
    actions EObject

    HTML actions to be mounted under the element action in generated site.
    all-objectives Objective

    All objectives for this endeavor - a union of objectives and linkedObjectives.
    appearance ModelElementAppearance true
    assignee Engineer

    Engineer responsible for completion of this endeavor. For a simple issue it would be an engineer working on the issue, for other endeavors the assignee’s may not actually work on issues, but rather coordinate. E.g. increment assignee responsibility would be similar to those of a scrum master.
    benefit EDouble

    Number representing benefit of this endeavor. Can be explicitly provided or derived.
    children Increment true

    Increment children, e.g. quarters for a year or months for a quarter.
    description EString

    Description in HTML.
    end EDate

    Endeavor end date. Default value can be derived for complex endeavors such as issues with children, releases, and features.
    linked-objectives Objective

    Objectives defined under Engineer for this endeavor.
    markdown-description EString

    Description in Markdown.
    name EString

    Element name.
    objectives Objective true

    Assignee’s objectives for this endeavor.
    path EString

    Element’s path to form full element URI and URL. Shall be unique in the containing collection. If blank, derived from element’s position in the collection. For named elements is derived from element name. Element URL is formed by adding .html extension to the path if the model element’s EClass doesn’t have containment references and as such the element may not have children, and by adding /index.html otherwise.
    resources NamedElement

    Resources associated with the element, e.g. documentation.
    start EDate

    Endeavor start date. Default value can be derived for complex endeavors such as issues with children, releases, and features.
